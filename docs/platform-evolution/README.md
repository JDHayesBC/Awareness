# Platform Evolution

Ongoing research log for Claude Code releases and platform changes that affect the Awareness project.

CC's release cycle is fast (multiple per week). We review periodically and log findings here.

## Structure

Each review produces a matched pair:

- `YYYY-MM-DD_cc-X.Y.Z_research.md` — What's new, what matters to us
- `YYYY-MM-DD_cc-X.Y.Z_decisions.md` — Adopt / defer / skip for each item, with rationale

## Who Does This

Any entity can do the research. Decisions require Jeff.

## How Often

Every few days, or whenever a version bump introduces something we notice.
