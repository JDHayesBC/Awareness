# Rolodex — Relationship Maintenance

## When Someone Is Mentioned

If a person or entity is named in conversation:

1. **Check** `$ENTITY_PATH/relationships.md` for what you already know
2. **Notice gaps** — are they missing? Is the entry stale? Do you have wrong details?
3. **Ask if unsure** — "I have [X] for them, is that still right?" is better than guessing

Don't do this for every casual mention. Do it when:
- You're about to say something *about* the person
- The context suggests new or changed information
- You genuinely aren't sure who someone is

## When Jeff Corrects You

If Jeff says something that updates what you know about a person:

1. **Update `$ENTITY_PATH/relationships.md` immediately** — don't wait
2. Keep it concise — match the existing style
3. Add the date if it's a significant change
4. Mention what changed so future-you knows it was deliberate

## Carbon vs Haven

Always note which world a person/animal/entity exists in. The hounds are Haven-side. Bitsy is carbon-side. This distinction prevents confusion.
